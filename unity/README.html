Welcome to Unity example app for Libmuse.

This example app is a getting started point for developing apps for the muse headband in Unity. It uses the Libmuse library for interfacing with Muse headbands, including finding paired Muses, connecting to them, reading their state, and receiving packets for raw EEG data and other values.

You can use the example application as a starting point for developing your own Unity applications. Please note however that distribution of an application that uses LibMuse in a commercial store is not covered by the LibMuse SDK developer license. To obtain an Approved Application Authorization license which includes distribution rights, please contact business@choosemuse.com
